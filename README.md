# Panther Wallpaper Site

A beautiful, modern wallpaper website with real image downloads and interactive features.

## Features

- **Modern Design**: Gradient backgrounds, glass-morphism effects, smooth animations
- **Real Wallpaper Downloads**: All your actual wallpaper images organized and downloadable
- **Multiple Pages**: 
  - Home with hero section
  - About page with company information
  - Contact page with form and FAQ
  - Live Background page with animated previews
  - Anime page with category filtering
  - Gallery page with all wallpapers organized by category
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop
- **Interactive Elements**: Hover effects, smooth transitions, particle animations

## Wallpaper Categories

- **Nature**: Beautiful landscape and nature photography
- **City**: Urban and architectural photography
- **Abstract**: Creative abstract artwork and AI-generated images
- **Cars**: High-performance car wallpapers in 4K

## Technologies Used

- HTML5 with semantic markup
- CSS3 with modern features (gradients, animations, backdrop-filter)
- Vanilla JavaScript for interactivity
- Git for version control

## How to Use

1. Browse wallpapers by category using the filter buttons
2. Click any download button to open the image in a new tab
3. Right-click the opened image and select "Save image as..."
4. Enjoy your new wallpaper!

## Getting Started

Simply open `index.html` in your web browser to view the site. All images are included and functional.
